# Knowledge Check System

The platform includes lightweight knowledge checks on key Healthcare Transformation Operating System pages.

## Purpose

Help visitors actively learn instead of passively browsing.

## Current pages

- Readiness Assessment
- Decision Journal
- Implementation Playbook
- Benefits Realization Tracker
- Medicine Support Hub
- Proof Room

## Design rule

Knowledge checks should be short, useful and professional. They should reinforce executive decision-making concepts without making the platform feel like a childish quiz.

## Privacy

Completion is stored locally in the visitor's browser. No account or backend storage is required.
