Automated checks

What the validator protects

Metadata

Every HTML page should have a title, meta description and canonical URL.

Structure

Every page should have exactly one H1 so search engines and screen readers understand the page.

Internal links

Internal links are checked against repository routes to catch broken navigation before merge.

Required scripts

Core assistant, enhancement and routing scripts are checked so the experience stays consistent.

Image alt text

Images should include alt text when they appear in HTML pages.

Sitemap coverage

Important pages should appear in the sitemap so discovery does not depend on luck.

Workflow

When the checks run

The validator runs on pull requests, pushes to master and manual workflow dispatch. If a quality gate fails, the fix should happen before the page is treated as complete.
