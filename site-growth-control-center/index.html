Weekly rhythm

The operating cadence

Monday

Publish one authority post

Use the LinkedIn Content Engine. Route readers to one page only.

Tuesday

Send targeted outreach

Use the Outreach Command System. Send 5 focused messages, not generic networking notes.

Wednesday

Improve one proof asset

Refine a case study, proof room entry, claims wording or executive brief section.

Thursday

Follow up with value

Send one useful resource based on audience: assessment, proof room, PMO article or operating system.

Friday

Review signals

Check which page and message produced attention. Repeat what worked next week.
