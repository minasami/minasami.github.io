The website is intentionally lightweight and framework-free:

- Responsive CSS
- Vanilla JavaScript
- JSON-LD structured data
- GitHub Pages hosting
- Google Analytics 4
- Automated HTML, link and Lighthouse checks through GitHub Actions


- Validates HTML

```text

```
