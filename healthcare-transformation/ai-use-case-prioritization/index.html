AI Use-Case Prioritization in Healthcare

AI use cases should be selected according to value, feasibility, workflow fit, data readiness and risk.

Scoring criteria

  • User need
  • Expected value
  • Data readiness
  • Technical feasibility
  • Workflow fit
  • Potential harm
  • Monitoring capability

Practical method

  1. Define the task or decision.
  2. Describe the current workflow.
  3. Assess available data.
  4. Classify the risk.
  5. Estimate effort and benefit.
  6. Select a controlled pilot.
