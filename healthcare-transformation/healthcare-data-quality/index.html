Healthcare Data Quality

Healthcare data quality means information is accurate, complete, timely, consistent and fit for its intended use.

Core dimensions

  • Accuracy
  • Completeness
  • Timeliness
  • Consistency
  • Validity
  • Uniqueness

Improvement cycle

  1. Identify critical data elements.
  2. Define quality rules.
  3. Measure the baseline.
  4. Find root causes.
  5. Assign owners and corrective actions.
  6. Monitor trends.
