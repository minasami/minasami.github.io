Patient Portal Delivery

Patient portal delivery should improve access, communication and task completion for users.

Priority capabilities

  • Identity and access
  • Appointments
  • Documents and results
  • Secure communication
  • Requests and follow-up
  • Accessibility and support

Delivery steps

  1. Define user needs.
  2. Map the current journey.
  3. Select the first useful capabilities.
  4. Prepare support and escalation.
  5. Pilot with representative users.
  6. Measure adoption and completion.
