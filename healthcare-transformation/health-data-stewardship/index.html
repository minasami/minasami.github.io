Health Data Stewardship

Data stewardship assigns responsibility for the quality, meaning, access and lifecycle of important healthcare information.

Steward responsibilities

  • Maintain definitions
  • Monitor quality
  • Resolve data issues
  • Review access needs
  • Coordinate changes
  • Document decisions

Operating cycle

  1. Identify critical data domains.
  2. Name accountable owners and stewards.
  3. Define quality rules.
  4. Review issues and exceptions.
  5. Track corrective actions.
