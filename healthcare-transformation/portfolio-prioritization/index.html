Healthcare Portfolio Prioritization

Portfolio prioritization helps leaders decide which initiatives should start, pause, sequence or stop.

Core criteria

  • Strategic alignment
  • Patient and operational value
  • Urgency
  • Readiness
  • Risk
  • Dependencies
  • Capacity
  • Expected benefits

Practical method

  1. Define a common scoring model.
  2. Score initiatives with cross-functional input.
  3. Review dependencies and resource conflicts.
  4. Separate mandatory work from discretionary work.
  5. Approve a balanced portfolio.
  6. Reassess regularly.
