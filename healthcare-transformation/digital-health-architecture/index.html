Digital Health Architecture

Digital health architecture connects user needs, workflows, applications, data, integrations and infrastructure.

Core layers

  • User experience
  • Workflow and business services
  • Applications
  • Data
  • Integration
  • Security and infrastructure

Design principles

  1. Start from outcomes and workflows.
  2. Use clear system boundaries.
  3. Reduce duplicate data entry.
  4. Plan integration early.
  5. Design for support and change.
