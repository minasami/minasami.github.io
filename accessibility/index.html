Accessibility Statement

This website aims to make Mina’s professional profile, case studies and contact information usable by as many people as possible.

Accessibility commitment

The site is being developed with practical accessibility principles in mind, including readable typography, responsive layouts, keyboard-accessible navigation, meaningful headings, descriptive link text and support for reduced-motion preferences.

Current accessibility features

  • A skip link on the homepage for keyboard users
  • Semantic headings, landmarks and navigation labels
  • Responsive layouts for mobile and desktop screens
  • Visible focus behavior supported by browser defaults
  • Alternative text for meaningful images and QR codes
  • Reduced-motion support where animations or transitions are used
  • Print-friendly recruiter and CV views

Known limitations

Some embedded or third-party content may not offer the same level of accessibility as the main website. This may include the live Medicine Support Hub preview, third-party QR generation, LinkedIn, GitHub, WhatsApp and Replit-hosted content.

Alternative access: the main case-study text, project descriptions and direct contact information remain available without using embedded previews.

Testing and improvement

The repository includes automated HTML validation, link checking and Lighthouse audits. These checks support ongoing improvement but do not replace testing with real users and assistive technologies.

Report an accessibility barrier

To report an issue, email jesussavedmina@gmail.com. Please include the page address, the problem encountered and the device or assistive technology used when possible.

Reasonable efforts will be made to provide the requested information in an accessible alternative format.
